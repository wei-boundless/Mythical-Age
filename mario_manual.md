# Super Mario Bros 世界 1-1 — 游戏说明书

## 概述

经典《超级马里奥兄弟》世界 1-1 的 HTML5 网页克隆版。在浏览器中打开 `mario.html` 即可游玩。

## 操作方式

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 向左移动 | `←` 或 `A` | 点击屏幕左 1/3 区域 |
| 向右移动 | `→` 或 `D` | 点击屏幕右 1/3 区域 |
| 跳跃 | `空格` / `↑` / `W` | 点击屏幕上 1/2 区域 |
| 下蹲（大马里奥时） | `↓` 或 `S` | 点击屏幕下 1/5 区域 |
| 重新开始 | `R`（游戏结束或通关后） | — |

## 关卡元素

### 地面砖块（棕色）
- 可站立行走的基础地形
- 部分区域有坑洞，掉落即死亡

### 砖块（棕色带纹路）
- 小马里奥：从下方撞击会弹回，无法破坏
- 大马里奥：从下方撞击可砸碎砖块，获得 50 分

### 问号砖块（黄色带 ?）
- 从下方撞击可获得随机奖励：
  - **金币**（40%）：+200 分
  - **红色蘑菇**（50%）：大马里奥状态下吃蘑菇无效，小马里奥吃后变大
  - **1-UP 绿色蘑菇**（10%）：增加一条命

### 隐藏砖块（隐形）
- 从下方撞击后出现，固定奖励金币 +200 分

### 管道（绿色）
- 装饰性障碍物，部分管道中有食人花敌人

### 旗杆（关卡终点）
- 到达旗杆即通关
- 剩余时间 ×10 计入最终得分

## 敌人图鉴

### 板栗仔（Goomba）
- 棕色蘑菇状敌人，在地面左右移动
- 从上方踩踏可消灭（+100 分）
- 侧面或下方碰到会使马里奥受伤

### 诺库龟（Koopa Troopa）
- 绿色龟壳敌人，在地面巡逻
- 踩踏后龟壳会弹出
- 踢动龟壳可消灭前方敌人（+200 分/个）
- 龟壳反弹回来会伤到马里奥

### 食人花（Piranha Plant）
- 从管道中周期性伸出缩回
- 无法踩踏消灭，碰到即受伤
- 建议跳过或绕行

## 道具系统

| 道具 | 效果 |
|------|------|
| **红色蘑菇** | 小马里奥吃后变大（+1000 分），可砸碎砖块、承受一次伤害 |
| **绿色 1-UP 蘑菇** | 增加一条命 |
| **金币** | 收集计数，每个 +200 分 |

## 计分规则

| 事件 | 分数 |
|------|------|
| 踩踏敌人 | 100 |
| 龟壳击中敌人 | 200 |
| 金币/隐藏砖 | 200 |
| 砸碎砖块 | 50 |
| 吃红色蘑菇 | 1000 |
| 通关时间奖励 | 剩余秒数 ×10 |

## 失败条件

- 掉落坑洞：损失一条命，关卡重置
- 被敌人侧面碰到：
  - 大马里奥 → 变小，短暂无敌
  - 小马里奥 → 死亡
- 时间归零：游戏结束
- 生命归零：游戏结束

## 通关条件

到达关卡最右侧的旗杆处即可通关。

## 技术信息

- 纯 HTML5 + Canvas 实现，无需外部依赖
- 像素风格渲染（3 倍缩放）
- 支持键盘和触屏操作
- 关卡为 Super Mario Bros 世界 1-1 的忠实复刻
