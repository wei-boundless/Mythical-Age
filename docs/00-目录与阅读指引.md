# 深入 Claude Code 源码：AI Agent 应用的全栈技术解析

> **从一个真实的 AI 产品源码中，学会构建 AI Agent 应用的全栈技术**

## 关于本书

本书是一本基于 Anthropic **Claude Code** 完整源码的深度技术分析书籍。Claude Code 是一个 AI 驱动的命令行编程助手，包含约 1900 个源码文件（其中 TypeScript 文件约 1884 个），涵盖了从 System Prompt 工程、多 Agent 编排、工具系统设计到终端 UI 渲染的完整技术栈。

我们将以「源码阅读」的方式，逐模块解析这个真实的生产级 AI 产品，从中提炼出可迁移到你自己项目的设计模式与工程实践。

### 目标读者

- 有 1-3 年经验的程序员，了解 TypeScript
- 对 AI 应用开发感兴趣，想从真实产品中学习
- 希望理解大型 AI CLI 产品的架构设计

### 阅读约定

- 正文使用中文，技术术语保留英文原文（如 Agent、Tool、System Prompt）
- 代码引用格式：`文件路径:行号范围`
- 每篇文章可独立阅读，但建议按顺序
