# 编排系统阶段收口

日期：2026-04-29  
定位：本文件用于冻结当前编排系统重构阶段的完成口径。收口范围必须包括 `OperationGate + directive-only executor` 前置合同；真实执行、真实写回和多智能体仍不开放。

---

## 1. 本阶段完成结论

当前阶段完成的是：

```text
single_agent preview control plane
OperationGate preflight contract
directive-only executor preflight contract
```

不是：

```text
real execution orchestration
multi-agent orchestration
memory write orchestration
commit / output orchestration
```

换言之，编排系统现在已经能形成完整的预览控制链，并且已经明确真实执行前必须经过 OperationGate、Executor 只能消费 RuntimeDirective。但当前仍不生成可消费 RuntimeDirective，不让 OperationGate 通过，不进入 Executor，不放行 CommitGate。

## 5. 明确暂停的事项

本阶段不继续推进：

```text
OrchestrationPlanPreview -> OrchestrationPlan adoption
ResourcePolicyPreview -> AdoptedResourcePolicy adoption
RuntimeDirectiveCandidate -> RuntimeDirective
OperationGate 真实通过
ToolExecutor / WorkerExecutor / AgentExecutor
ResultCandidate
OutputBoundary
真实 CommitGate
multi-agent topology adoption
```

原因：

```text
记忆系统尚未重构。
上下文管理尚未重构。
OutputBoundary 尚未重构。
query 旧执行能力仍在迁移期。
过早放行真实执行会让资源、记忆、写回边界重新耦合。
```

## 6. 与后续系统的接口约定

### 6.1 与记忆系统

编排系统只接收：

```text
MemoryContextCandidate
MemoryRestoreCandidate
MemoryIntentCandidate
MemoryWriteCandidate
MemoryPolicyCandidate
```

编排系统不直接：

```text
读取 durable memory 文件。
写 durable memory。
修改 session-memory。
决定哪些记忆进入 PromptManifest。
```

这些职责归 MemorySystem / ContextPolicy / CommitGate。

### 6.2 与上下文管理

编排系统只引用：

```text
context_package_ref
prompt_manifest_ref
memory_context_ref
task_prompt_contract_ref
```

编排系统不拼 prompt，不压缩上下文，不直接注入 system prompt。

### 6.3 与执行层

未来执行层只接受：

```text
RuntimeDirective
```

不接受：

```text
RuntimeDirectiveCandidate
QueryExecutionPlan
query_understanding.tool_name
worker_plan
TaskBinding preview
ResourcePolicyPreview
```

### 6.4 与写回层

未来写回只接受：

```text
CommitCandidate
CommitGate decision
```

不接受：

```text
QueryRuntime direct persistence
memory write side effect
session message append bypass
```

## 7. 下一阶段入口

下一阶段不是继续打开执行，而是重构记忆系统。

记忆系统重构完成前，编排系统只允许做小范围维护：

```text
修 preview 回归。
补 trace 字段。
清理 query 旧执行入口。
OperationGate/executor preflight 回归维护。
保持 fail-closed。
```

记忆系统重构完成后，再进入：

```text
Read-only RuntimeDirective trial
ResultCandidate / OutputBoundary
CommitGate real decision
```
