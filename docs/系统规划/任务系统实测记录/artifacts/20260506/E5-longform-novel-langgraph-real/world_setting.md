# 洪荒时代 —— 世界观设定总纲

## 一、世界本源

洪荒时代，天地初分未久，万物尚在成形之中。世界并非由单一神祇创造，而是由五种原初力量各自演化、彼此交织而成。这五种力量分别对应五方：东之青木、中土之水、西之山石、南之火焰、北之玄夜。它们不是人格化的神，而是洪荒本身——是大地、河流、山脉、火焰与夜幕的意志。

五方力量各自独立，互不统属，也互不敌对。它们之间的关系更接近自然界的相生相克：水润木生，火炼金石，夜幕笼罩万物。没有善恶阵营，没有光明与黑暗的二元对立，只有力量与力量之间的自然博弈。

## 二、五方格局

### 东荒：青木之域
- **核心力量**：勾芒——洪荒中最繁荣的青木，携带东风、青烟与万物萌发的记忆。
- **地貌特征**：无尽林海、藤蔓古路、雾霭深谷。植被覆盖一切，城市与聚落建于巨树之上或根系之间。
- **众生形态**：东荒众生以采集、驯化、与草木共生为生。他们懂得倾听植物的记忆，能从年轮中读取远古的故事。
- **核心法则**：萌发与循环。东荒相信万物有灵，生死不过是形态转换，枯木中藏着新芽。
- **记忆特性**：青木记忆最古老、最绵长，如同年轮层层叠加。触碰青木记忆需要耐心——它不会一次性展现全部，而是像树木生长一样缓慢释放。代价是"缠绕"：触碰者可能被记忆中的根系缠住，难以脱身。

### 中土：水府之域
- **核心力量**：河伯——洪荒中最神圣的河流，携带百川、渡口与古老祭辞的记忆。
- **地貌特征**：水网密布，大河流经全境，支流如血脉延伸至四方。渡口是聚落的中心，也是信息与贸易的节点。
- **众生形态**：中土众生以渔猎、航运、祭祀为生。他们是洪荒中最擅长记录与传承的族群，祭辞中保存着比任何族群都古老的知识。
- **核心法则**：汇聚与记忆。水流不腐，记忆不灭。每一条汇入大河的水流都携带着它所经历的一切。
- **记忆特性**：祭辞记忆最庞杂、最沉重。河流不分清浊，一切记忆都被收纳。触碰祭辞记忆意味着面对海量信息——其中大部分是平凡的、重复的、甚至痛苦的。代价是"淹没"：触碰者可能被记忆的洪流冲散自我意识。

### 西荒：山城之域
- **核心力量**：四岳——洪荒中最巍然的山脉，承载地脉、聚落与万城之盟的记忆。
- **地貌特征**：群山连绵，矿脉丰富。城市建于山腹与峰顶之间，以石桥和隧道相连。万城之盟是西荒最庞大的政治实体。
- **众生形态**：西荒众生以采矿、冶炼、建筑为生。他们崇尚秩序与契约，万城之盟的律法刻于石碑，千年不易。
- **核心法则**：承载与盟约。山不轻移，盟不轻毁。四岳的记忆中刻着每一份盟约的原文与每一次背盟的代价。
- **记忆特性**：盟约记忆最坚固、最清晰。山石不腐，刻痕不灭。触碰盟约记忆相对安全——信息明确，结构清晰。但代价是"固化"：触碰者可能被盟约的逻辑束缚，失去变通的能力。

### 南荒：火庭之域
- **核心力量**：祝融——洪荒中最炽烈的火焰，携带光焰、锻造与人间烈火的记忆。
- **地貌特征**：火山群、熔岩平原、地热裂谷。锻造是南荒最核心的技艺，火庭既是工坊也是权力中心。
- **众生形态**：南荒众生以锻造、征战、开拓为生。他们崇尚力量与变革，认为停滞即是腐朽，火焰的意义在于烧尽旧物、锻造新器。
- **核心法则**：燃烧与重塑。火焰不保留记忆，它只负责烧尽与重塑。每一次燃烧都是新的开始。
- **记忆特性**：火焰记忆最短暂、最暴烈。火焰不保存，只转化。触碰火焰记忆极为困难——记忆在触碰的瞬间就被烧尽或重塑。主角的能力在南荒最弱，因为这里几乎没有"记忆"可以触碰，只有正在燃烧的当下。代价是"焚毁"：触碰者自身的记忆可能被火焰吞噬。

### 北荒：玄宫之域
- **核心力量**：玄女——洪荒中最神秘的夜幕，携带月辉、星图与渊深通玄的记忆。
- **地貌特征**：永夜苔原、冰封深海、星辉穹顶。玄宫并非固定建筑，而是随星象变化而移动的神秘领域。
- **众生形态**：北荒众生以观星、占卜、守夜为生。他们是洪荒中最沉默也最深邃的族群，懂得从星图中读取命运的轨迹。
- **核心法则**：静观与通玄。夜幕不言语，但它覆盖一切、看见一切。玄女的记忆中藏着所有被白昼遗忘的秘密。
- **记忆特性**：星图记忆最深不可测、最危险。夜幕覆盖一切，星图记录一切——包括尚未发生的事。触碰星图记忆意味着面对命运本身。代价是"吞噬"：星图可能吞噬触碰者的记忆、身份甚至存在。主角的能力在北荒最强也最危险——这里记忆最丰富，但代价也最高。

## 四、五方关系

五方之间不存在统一政权，也没有全面战争。关系以"边界"为核心概念：

- **东与中**：林海与水网在边界处交汇，形成湿地与沼泽。两方众生在此混居，文化交融最深。
- **中与西**：大河发源于西山，中土与西荒因水源而天然联结。万城之盟的律法中有一条古老条款：永不阻断河源。
- **西与南**：山脉与火山交界处是洪荒最活跃的地质带。西荒的矿石运往南荒锻造，南荒的兵器输往西荒装备城防。
- **南与北**：火焰与夜幕几乎不相交。南荒众生畏惧北荒的永夜，北荒众生回避南荒的炽热。边界是一片无人区。
- **北与东**：夜幕笼罩林海边缘时，东荒众生相信那是青木沉睡的时刻。两方之间有一种沉默的默契，互不侵扰。

## 五、独立主角的生存空间

五方格局为独立主角提供了明确的生存空间与冲突土壤：

- 主角不属于五方中的任何一方核心，但可以在边界地带生存、穿行、成长。
- 五方力量不是主角的盟友或敌人，而是主角必须面对的世界条件。
- 主角的旅程可以跨越五方，每一次跨越都意味着面对不同的自然法则与社会规则。
- 五方力量本身不会主动介入主角的命运——它们太大了，大到不会注意一个个体。但主角的行动可能触发五方力量的回应。

## 六、设定边界

- 本总纲不预设完整世界规则。五方内部的具体制度、历史事件、关键人物留待人物设定总表和剧情设定总纲补充。
- 本总纲不预设主线冲突。冲突将从人物设定与剧情设定中自然生长。
- 本总纲不预设主角姓名、身份、能力、命运。主角设定由人物设定总表独立完成。
- 五方力量（勾芒、河伯、四岳、祝融、玄女）不是角色，不是反派，不是导师，不是任务发布者。它们是世界本身的一部分，如同重力、气候、昼夜。
