# 《洪荒时代》第一卷卷纲

## 输入引用声明

本卷纲的规划依据来自以下上游输入文件：

- **novel_bible.md**：提供世界观设定、人物设定、主线冲突框架、伏笔体系、各卷功能定位等核心创作约束。

本卷纲承接上述输入，将第一卷定位为**"洪荒初醒·主角启程"**，完成从世界铺陈到主角确立、从散点事件到主线收束的第一阶段叙事。

## 第一卷目标

### 核心目标
第一卷（约20万字，40章）完成以下叙事任务：

1. **世界建立**：五荒格局初步呈现——东荒青木之域、中土水府流域、西荒万城之盟、南荒火庭锻域、北荒玄宫星渊，各自以侧写或局部介入方式进入叙事视野。
2. **主角确立**：独立主角（非五条背景本身）从微末中出场，完成身份锚定、初始动机建立、第一次关键选择。
3. **主线引燃**：五荒之间的深层矛盾露出第一道裂隙，主角被卷入其中，形成第一卷末的不可回避的推动力。
4. **伏笔体系启动**：至少埋设8条跨卷伏笔，其中3条在第一卷内完成回收，5条明确指向后续卷。

### 情感锚点
- 读者在第一卷结束时，应清楚主角是谁、想要什么、面临什么。
- 对五荒世界有具象感知，对至少两个荒域产生"想继续看"的期待。

- 主角形象是否清晰可感

- 第二卷承接点是否清晰有力

## 风险与注意事项

1. **五荒平衡**：第一卷需让五条背景种子全部落地并各有深入章节，但需避免平均用力导致的节奏松散。河伯线与四岳线作为第一卷重点深入对象，祝融线与玄女线次之，勾芒线以意象渗透为主。
2. **主角独立性**：主角必须始终保持独立人格和独立选择，不能成为五荒的传声筒或工具人。
3. **信息密度控制**：第一卷承担大量世界观建立任务，需注意"展示而非告知"，避免信息倾泻。
4. **伏笔可管理性**：8条跨卷伏笔需在后续卷中持续追踪，第一卷内需确保每条伏笔的投放都有明确的后续指向。
5. **终章力度**：第40章作为第一卷终章，需同时满足"阶段性满足感"和"继续阅读的驱动力"，是难度最高的章节之一。
