# 洪荒时代 —— 剧情设定总纲

## 一、故事核心

### 一句话梗概
一个不属于五方任何一方的"无属之人"，在触碰洪荒五方远古记忆的旅程中，寻找属于自己的位置。

### 核心冲突
主角能够感知五方力量记忆的罕见能力，既是天赋也是诅咒。五方众生对此能力的反应各不相同——有人恐惧、有人觊觎、有人否定、有人想利用。主角必须在这些反应中穿行，同时承受每一次触碰记忆带来的代价，最终在五方之间开辟一条前所未有的道路。

### 主题
- **归属与自由**：归属是否意味着放弃自由？自由是否意味着永远漂泊？
- **记忆与代价**：触碰世界的记忆需要付出什么？有些记忆是否不该被触碰？
- **个体与世界**：一个个体能否在不属于任何一方的情况下，改变世界的格局？

## 二、五卷结构

### 第一卷：边界之芽（约200,000字）
**卷主题**：发现与出发

**主线**：
- 主角在边界湿地过着边缘生活，偶然间触碰到一段青木的年轮记忆，首次意识到自己的能力。
- 渡教会主角生存技能，但无法回答主角关于"归属"的问题。
- 芽发现主角的能力异常，警告主角这种能力的危险性，但主角无法抑制触碰记忆的冲动。
- 烬出现，开始追猎主角——他认为主角的能力会打破五方平衡。
- 主角被迫离开边界，踏上穿越五方的旅程。第一卷以主角跨出边界、面对未知的洪荒世界结束。

**关键事件**：
1. 湿地日常：主角的边界生活与第一次记忆触碰
2. 渡的教导：生存技能与无解之问
3. 芽的发现：年轮解读中的异常
4. 芽的警告：触碰记忆的代价
5. 烬的降临：追猎开始
6. 离开边界：主角的选择与第一卷终局

**人物弧线**：
- 主角：从被动生存到主动出发
- 渡：从陪伴到分离
- 芽：从好奇到恐惧
- 烬：追猎者登场

### 第二卷：水与石（约200,000字）
**卷主题**：深入与理解

**主线**：
- 主角进入中土水府，触碰河流的祭辞记忆，了解到洪荒最古老的知识。
- 在中土遇到祭辞守护者，开始理解记忆的真正分量——祭辞中记录的不是荣耀，而是无数被遗忘的牺牲。
- 追猎者烬的追捕升级，主角被迫继续向西，进入西荒山城之域。
- 在西荒，主角触碰到山脉的盟约记忆，发现万城之盟并非铁板一块——古老的背盟记录中藏着正在酝酿的新危机。
- 主角开始意识到：五方记忆不只是过去，它们也指向未来。

**关键事件**：
1. 大河之上：触碰祭辞记忆
2. 祭辞守护者：记忆的重量
3. 渡口的告别：与过去的切割
4. 进入西荒：山城与盟约
5. 背盟的记忆：发现隐藏的危机
6. 烬的逼近：追猎升级

**人物弧线**：
- 主角：从触碰记忆到理解记忆的代价
- 祭辞守护者：从守护到动摇
- 盟约仲裁者：从秩序维护者到危机发现者
- 烬：追猎者的信念开始出现裂痕

### 第三卷：火炼（约200,000字）
**卷主题**：暴露与选择

**主线**：
- 主角的能力被更广泛的势力发现。西荒盟约中的某一派想利用主角触碰记忆的能力来揭露对手的背盟历史；南荒火庭则视主角为必须消灭的威胁。
- 主角被迫进入南荒——这是最危险的一方，因为火焰不保留记忆，只负责烧尽。主角的能力在这里最弱，也最危险。
- 在南荒，主角遇到另一个能触碰记忆的人——但这个人已经被记忆吞噬，成为行尸走肉。这是主角第一次直面能力的终极代价。
- 烬的背景被揭示：他曾经追猎过另一个类似的人，失败导致了那个人的毁灭和烬自己的流放。他对主角的恐惧源于创伤而非恶意。
- 主角必须在各方势力的拉扯中做出选择：成为工具、成为牺牲品，还是找到第三条路。

**关键事件**：
1. 暴露：能力被各方知晓
2. 西荒的博弈：被利用与反利用
3. 进入南荒：最危险的一方
4. 另一个触碰者：能力的终极代价
5. 烬的真相：追猎者的创伤
6. 选择：第三条路的雏形

**人物弧线**：
- 主角：从被追逐到主动选择
- 烬：从追猎者到复杂对手（可能转向）
- 另一个触碰者：警示与镜像
- 各方代理人：利益与恐惧的交织

### 第四卷：夜幕（约200,000字）
**卷主题**：代价与蜕变

**主线**：
- 主角逃入北荒——五方中最神秘的一方。永夜苔原与星辉穹顶是主角从未经历过的环境。
- 在北荒玄宫，主角遇到观星者——一个终生研究星图却从未离开北荒的老人。观星者不关心五方政治，只关心星图中预示的"变局"。
- 主角决定触碰最危险的记忆——玄女的星图记忆。这是五方中最深不可测的记忆，触碰它的代价可能是失去自我。
- 触碰星图记忆后，主角看到了洪荒的全貌——不是过去，而是五方力量交织成的命运之网。主角在这张网中看到了自己的位置：不是归属于某一方，而是成为五方之间的连接点。
- 代价显现：主角的一部分记忆被星图吞噬。主角失去了某些珍贵的东西——可能是对某个人的记忆，可能是自己的一部分过去。
- 烬在第四卷做出关键选择：继续追猎，还是与主角并肩。

**关键事件**：
1. 进入北荒：永夜中的旅程
2. 观星者：星图与变局
3. 触碰星图：最危险的记忆
4. 命运之网：看到全貌
5. 代价：失去的记忆
6. 烬的选择：追猎者的终局

**人物弧线**：
- 主角：从承受代价到主动付出代价
- 观星者：从旁观到介入
- 烬：最终抉择
- 主角失去的记忆：为第五卷埋下伏笔

### 第五卷：归墟（约200,000字）
**卷主题**：开辟与归属

**主线**：
- 主角带着星图中看到的全貌回到五方交界之地——不是边界湿地，而是一个更中心的位置。
- 五方之间的隐藏危机在第四卷的伏笔中爆发：西荒盟约面临破裂，中土祭辞出现断裂，东荒年轮开始枯萎，南荒火焰失控蔓延，北荒星图出现从未有过的异象。
- 主角意识到：五方记忆中的危机不是孤立的，它们彼此关联。而主角——作为唯一触碰过五方全部记忆的人——是唯一能看到全貌的人。
- 主角不选择拯救任何一方，也不选择毁灭任何一方。主角选择成为"归墟"——一个不属于五方但连接五方的存在，让五方力量通过主角的记忆彼此看见。
- 最终，主角找到了自己的归属：不是加入某一方，而是成为五方之间的桥梁。这个位置从未存在过，是主角用自己的代价开辟出来的。

**关键事件**：
1. 归来：回到五方交界之地
2. 危机爆发：五方同时异变
3. 全貌：只有主角能看到关联
4. 归墟：开辟新的存在方式
5. 代价的终局：主角最终失去了什么，又得到了什么
6. 尾声：洪荒的新格局

**人物弧线**：
- 主角：从寻找归属到创造归属
- 五方众生：从各自为政到被迫面对关联
- 烬（如果存活）：从追猎者到见证者
- 所有核心配角的终局

## 三、伏笔与呼应系统

### 跨卷伏笔
| 伏笔 | 埋设卷 | 揭示卷 | 内容 |
|------|--------|--------|------|
| 主角能力的真正来源 | 卷一 | 卷四/五 | 为什么是主角？ |
| 渡的真实身份 | 卷一 | 卷五 | 渡不只是普通船夫 |
| 芽看到的年轮异象 | 卷一 | 卷五 | 东荒枯萎的预兆 |
| 背盟记录中的名字 | 卷二 | 卷五 | 与当前危机的关联 |

| 星图中的变局 | 卷四 | 卷五 | 五方异变的本质 |

### 情感伏笔
- 主角在卷四失去的记忆，在卷五以另一种形式回归
- 烬与主角的关系从猎杀到理解，构成全书最复杂的人物关系线
- 渡在第一卷说的某句话，在第五卷被主角重新理解

## 四、节奏控制

### 每卷节奏
- **开篇**（前10%）：建立卷内新环境与新关系
- **发展**（10%-40%）：触碰记忆，展开卷内冲突
- **转折**（40%-60%）：卷内中点转折，揭示新信息
- **升级**（60%-90%）：冲突升级，代价显现
- **收束**（90%-100%）：卷内收束，为下一卷埋钩

### 全书节奏
- 卷一：慢热，建立世界与人物
- 卷二：加速，深入两方，信息密度提升
- 卷三：高潮前奏，冲突白热化
- 卷四：深潜，最黑暗也最深邃
- 卷五：爆发与收束，所有线索汇聚

## 五、设定边界

- 本总纲不预设章节正文。章节划分与具体场景留待卷规划阶段。
- 本总纲不锁定配角最终命运。人物终局留待写作过程中根据故事自然生长确定。
- 本总纲中的事件描述为方向性框架，具体情节可在写作中调整。
- 五方力量（勾芒、河伯、四岳、祝融、玄女）不作为剧情角色出现，它们的影响通过记忆触碰和世界异变间接体现。
