# 洪荒时代 —— 人物设定总表

## 一、人物设定原则

1. **独立主角**：主角不是勾芒、河伯、四岳、祝融、玄女这五条背景本身，也不是五方力量的化身或代言人。主角是一个独立的个体，在五方格局中寻找自己的道路。
2. **五方力量不是角色**：勾芒、河伯、四岳、祝融、玄女是世界的自然力量，不具人格，不出场对话，不发布任务，不充当导师或反派。它们如同气候与地质，是主角必须面对的条件而非互动的角色。
3. **配角从世界土壤中生长**：所有配角来自五方众生，有各自的出身、立场、欲望与局限。没有全知全能的智者，没有纯粹善恶的角色。
4. **人物服务于故事**：每个人物的存在都有叙事功能，不为填充而设。

## 二、独立主角

### 基础信息
- **姓名**：暂未命名（由后续章节写作中自然产生或由用户指定）
- **出身**：边界地带——东荒与中土交界的湿地沼泽。这里不属于任何一方核心区域，是五方文化交汇最深的边缘地带。
- **初始状态**：主角在故事开始时是一个"无属之人"——不属于五方中的任何一方，没有族群归属，没有固定居所，没有明确的身份标签。

### 核心设定
- **主角的独特之处**：主角拥有一种罕见的能力/特质——能够感知五方力量的"记忆"。不是与五方力量对话，而是能在特定条件下触碰到它们所承载的远古记忆：青木的年轮记忆、河流的祭辞记忆、山脉的盟约记忆、火焰的锻造记忆、夜幕的星图记忆。这种能力在洪荒中极为罕见，也极为危险——因为触碰这些记忆意味着触碰世界最原始的力量。
- **主角的局限**：这种能力不是控制力。主角不能命令五方力量做任何事，不能借用它们的力量战斗，不能通过它们预知未来。主角只能感知、理解、承受。每一次触碰记忆都会留下代价——身体上的、精神上的，或命运上的。
- **主角的驱动力**：寻找归属。作为一个"无属之人"，主角渴望知道自己属于哪里。这种渴望不是被动的伤感，而是主动的追寻——穿越五方，触碰记忆，在世界的碎片中拼凑自己的位置。

### 成长弧线（五卷框架）
- **第一卷**：从边界出发，初步发现自己的能力，面对"无属之人"的身份困境。
- **第二卷**：深入一方或两方，开始理解五方记忆的代价与意义。
- **第三卷**：能力被外界发现，成为各方争夺或恐惧的对象，面临选择。
- **第四卷**：触碰最危险的记忆，付出重大代价，重新定义自己与世界的关系。
- **第五卷**：找到自己的位置——不是归属于某一方，而是在五方之间开辟属于自己的道路。

## 三、核心配角

### 第一卷核心配角

**1. 边界引路人（暂名：渡）**
- 出身：中土水府边界渡口的老船夫
- 功能：主角的第一位引路人，教会主角在边界生存的实用技能
- 性格：沉默寡言，手上全是拉纤的老茧，对五方之事不置评
- 局限：他只能教主角怎么活下去，不能教主角怎么找到归属——因为他自己也没有答案
- 与主角关系：类似师徒但不正式，在第一卷后期因故分离
- **跨卷伏笔**：渡的真实身份在第五卷揭示——他不只是普通船夫（具体内容留待卷规划阶段细化）

**2. 东荒追忆者（暂名：芽）**
- 出身：东荒青木之域的年轮解读人
- 功能：第一个发现主角能力异常的人，也是第一个警告主角这种能力危险的人
- 性格：温和但偏执，毕生致力于解读一棵古木的年轮记忆
- 局限：她只关心东荒的记忆，无法理解主角为什么要触碰五方
- 与主角关系：短暂的同行者，在主角决定离开东荒时产生分歧
- **跨卷伏笔**：芽在第一卷看到的年轮异象，是第五卷东荒枯萎的预兆

**3. 追猎者（暂名：烬）**
- 出身：南荒火庭的流放者
- 功能：第一卷至第四卷的主要对抗力量，代表"外界对主角能力的恐惧"
- 性格：冷酷高效，曾是火庭最优秀的追猎者，因一次失败被流放
- 动机：他认为主角的能力是对世界秩序的威胁，必须在其成长之前消灭
- 与主角关系：猎人与猎物，但烬并非纯粹的恶——他的恐惧有合理根源
- **跨卷伏笔**：烬曾追猎过另一个能触碰记忆的人，那次失败导致了那个人的毁灭和烬的流放（卷三揭示）。烬在卷四面临最终抉择：继续追猎还是与主角并肩。

### 第二卷核心配角

**4. 祭辞守护者（暂名：沧）**
- 出身：中土水府祭辞传承世家
- 功能：引导主角理解祭辞记忆的真正分量
- 性格：庄重而疲惫，背负着世代传承的重担
- 局限：他守护祭辞太久，已经分不清哪些是祭辞中的记忆、哪些是自己的记忆
- 与主角关系：导师与警示——他让主角看到触碰记忆的长期代价

**5. 盟约仲裁者（暂名：磐）**
- 出身：西荒万城之盟的仲裁庭
- 功能：揭示西荒盟约体系中的裂痕与隐藏危机
- 性格：严谨到刻板，但内心深处对盟约的公正性产生了怀疑
- 局限：他发现了背盟记录中的异常，但体制内的身份让他难以公开质疑
- 与主角关系：信息提供者——他需要主角的能力来验证自己的怀疑

### 第三卷核心配角

**6. 另一个触碰者（暂名：烬痕）**
- 出身：未知（已被记忆吞噬，无法追溯）
- 功能：主角的镜像与警示——展示触碰记忆能力的终极代价
- 状态：已被记忆吞噬，成为行尸走肉般的存在。偶尔闪现清醒的碎片，但随即被新的记忆淹没
- 与烬的关系：烬曾经追猎过的人。那次追猎的失败——烬痕被记忆吞噬而非被烬捕获——导致了烬的流放
- 与主角关系：警示。主角在烬痕身上看到了自己可能的未来
- **跨卷伏笔**：烬痕在卷四的星图记忆中再次出现——星图保留了他被吞噬前的最后一段完整记忆，揭示了与主角的隐秘联系

### 第四卷核心配角

**7. 观星者（暂名：朔）**
- 出身：北荒玄宫，终生未离开北荒
- 功能：引导主角理解星图记忆的本质，帮助主角在付出代价后重新站立
- 性格：超然而温和，不关心五方政治，只关心星图中预示的"变局"
- 局限：他懂得解读星图，但从未触碰过星图记忆——他选择了安全的距离
- 与主角关系：最后的引路人。在主角触碰星图记忆并付出代价后，朔帮助主角从失去中恢复

### 第五卷核心配角

（第五卷核心配角将在卷规划阶段根据前四卷发展确定，此处仅预留位置）

## 四、人物关系网络（初始状态）

```
主角（无属之人）
  ├── 渡（边界引路人）── 教导生存，分离 → 卷五揭示真实身份
  ├── 芽（东荒追忆者）── 发现能力，分歧 → 卷五年轮异象
  ├── 烬（追猎者）────── 追猎，恐惧 → 卷四最终抉择
  ├── 沧（祭辞守护者）── 引导，警示
  ├── 磐（盟约仲裁者）── 信息提供，相互利用
  ├── 烬痕（另一个触碰者）── 镜像，警示 → 卷四星图揭示
  └── 朔（观星者）────── 最后引路人，恢复陪伴
```

## 五、设定边界

- 主角姓名暂不锁定，可在后续写作中确定或由用户指定。
- 配角姓名均为暂名，最终命名权归后续写作阶段。
- 本表不预设任何角色的最终命运，只设定初始状态与功能方向。
- 五方力量（勾芒、河伯、四岳、祝融、玄女）不出现在人物表中，它们不是角色。
- 人物数量遵循"少而精"原则，每卷核心配角不超过3人，避免百万字后期人物膨胀。
