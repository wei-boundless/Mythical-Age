# 第一卷《边界之芽》卷纲

## 卷信息
- 卷序：第一卷
- 卷名：边界之芽
- 字数分配：约 200,000 字
- 章节数：20 章（每章约 10,000 字）
- 主线定位：主角觉醒与边界初探

## 卷级主线
主角黎在五方洪荒的边界地带——东荒与中土交界的"芽镇"——意外触碰到勾芒遗留的一枚青木之种，从此开始感知到五方记忆碎片。与此同时，五方势力各自察觉到边界异动，分别派出使者探查。黎必须在五方使者的交错视线中找到自己的位置，同时面对青木之种带来的不可逆变化。

## 章节规划

### 第1章：芽镇之土
- 字数：约 10,000 字
- 主线功能：建立主角日常，引入芽镇世界观基础，埋下青木之种伏笔
- 关键事件：
  1. 黎在芽镇的清晨日常——采药、与镇民互动
  2. 芽镇的地理位置交代：东荒与中土交界，三不管地带
  3. 黎发现一株从未见过的青芽，触碰后短暂昏迷
  4. 醒来后发现自己能"听见"植物的低语
- 伏笔植入：青木之种（勾芒记忆碎片载体）、芽镇的特殊地理位置
- 出场人物：黎（主角）、老药农（配角，芽镇长者）
- 节奏控制：慢热引入，以日常铺陈为主，结尾抛出异能觉醒钩子

### 第2章：东来的风
- 字数：约 10,000 字
- 主线功能：东荒使者登场，黎的异能初步显现
- 关键事件：
  1. 黎试图隐藏新能力，但无法控制植物低语的涌入
  2. 东荒使者"青女"抵达芽镇，声称感应到勾芒气息
  3. 黎与青女首次接触，青女察觉黎身上的异常
  4. 黎从青女口中第一次听说"勾芒"之名
- 伏笔植入：青女的身份与使命、东荒对勾芒记忆的态度
- 出场人物：黎、青女（东荒使者）、老药农
- 节奏控制：外部势力介入，日常被打破，悬念上升

### 第3章：渡口之辞
- 字数：约 10,000 字
- 主线功能：中土势力登场，芽镇成为多方关注焦点
- 关键事件：
  1. 中土水府的巡河使"渡"抵达芽镇附近的古渡口
  2. 渡在渡口举行古老祭辞仪式，黎暗中观察
  3. 黎被渡发现，渡对黎身上的气息表现出审慎的兴趣
  4. 青女与渡在芽镇相遇，两方使者之间的微妙对峙
- 伏笔植入：河伯祭辞与勾芒记忆的关联、中土水府的态度
- 出场人物：黎、青女、渡（中土水府巡河使）
- 节奏控制：三方势力初步交汇，张力建立

### 第4章：青木之语
- 字数：约 10,000 字
- 主线功能：黎深入探索青木之种的秘密，首次主动使用能力
- 关键事件：
  1. 黎独自回到发现青芽的地点，尝试与青木之种深度沟通
  2. 进入"记忆之流"——体验勾芒残留的东荒记忆片段
  3. 记忆片段揭示：勾芒并非消失，而是将记忆分散藏于五方边界
  4. 黎从记忆之流中苏醒，发现青木之种已融入体内
- 伏笔植入：五方记忆分散的真相、黎作为"容器"的宿命
- 出场人物：黎（独白/记忆体验为主）
- 节奏控制：内省章节，信息量密集，世界观深度展开

### 第5章：西来的山影
- 字数：约 10,000 字
- 主线功能：西荒势力登场，四方势力在芽镇形成脆弱平衡
- 关键事件：
  1. 西荒山盟的使者"岩"抵达芽镇，带来四岳的口信
  2. 岩的态度强硬，要求芽镇接受西荒的"保护"
  3. 青女与渡联合抵制岩的扩张意图
  4. 黎在三人对峙中被迫暴露自己的能力
- 伏笔植入：四岳的万城之盟与边界扩张、三方制衡格局
- 出场人物：黎、青女、渡、岩（西荒山盟使者）
- 节奏控制：冲突升级，外部压力推动主角成长

### 第6章：火与夜
- 字数：约 10,000 字
- 主线功能：南荒与北荒势力同时抵达，五方齐聚芽镇
- 关键事件：
  1. 南荒火庭使者"炽"与北荒玄宫使者"冥"同日抵达
  2. 炽的热情与冥的沉默形成鲜明对比
  3. 五方使者首次齐聚，芽镇成为五方博弈的微缩舞台
  4. 黎被五方使者各自试探，每个人都想确认黎与勾芒的关系
- 伏笔植入：祝融与玄女对勾芒记忆的不同态度、五方平衡的脆弱性
- 出场人物：黎、青女、渡、岩、炽（南荒火庭使者）、冥（北荒玄宫使者）
- 节奏控制：人物群像展开，多方线索收束于芽镇

### 第7章：芽之抉择
- 字数：约 10,000 字
- 主线功能：黎在五方压力下做出第一次主动选择
- 关键事件：
  1. 五方使者各自向黎提出邀请/要求
  2. 黎意识到自己不能依附任何一方——青木之种的记忆属于五方共有
  3. 黎公开宣布：自己将留在芽镇，作为五方记忆的中立守护者
  4. 五方使者反应各异：青女默许、渡尊重、岩不满、炽好奇、冥沉默
- 伏笔植入：黎的"中立"立场为后续冲突埋下伏笔
- 出场人物：黎、五方使者、老药农
- 节奏控制：第一卷中段转折，主角从被动转为主动

### 第8章：记忆之根
- 字数：约 10,000 字
- 主线功能：黎开始系统性地探索体内青木之种的记忆
- 关键事件：
  1. 黎在老药农的帮助下建立冥想仪式
  2. 深入记忆之流，分别触碰五方记忆碎片
  3. 发现记忆之间存在"裂缝"——某些关键记忆被刻意抹去
  4. 在记忆裂缝中感知到一个模糊的"第六存在"
- 伏笔植入：被抹去的记忆、神秘的第六存在（全书终极伏笔）
- 出场人物：黎、老药农
- 节奏控制：信息深化，世界观扩展，终极悬念首次浮现

### 第9章：边界之试
- 字数：约 10,000 字
- 主线功能：外部威胁首次出现，黎的能力接受实战考验
- 关键事件：
  1. 芽镇遭遇"荒兽"袭击——一种不属于五方体系的异形生物
  2. 黎在危急中本能地调用青木之力，成功击退荒兽
  3. 五方使者对荒兽的出现感到震惊——荒兽本应只存在于洪荒之前的时代
  4. 黎的能力被五方正式确认：他不仅是记忆容器，更是勾芒之力的继承者
- 伏笔植入：荒兽的来源（与第六存在关联）、勾芒之力的战斗形态
- 出场人物：黎、五方使者、老药农、芽镇镇民
- 节奏控制：动作高潮，能力觉醒完成，外部威胁引入

### 第10章：芽之宣言
- 字数：约 10,000 字
- 主线功能：第一卷收束，黎确立自己的道路，五方格局初步形成
- 关键事件：
  1. 荒兽事件后，五方使者各自向本境汇报，芽镇地位升级
  2. 黎召集五方使者，正式宣布"芽镇协议"：芽镇作为五方中立地带
  3. 五方使者各自接受/保留态度，芽镇获得临时性五方认可
  4. 黎独自站在芽镇边界，感知到远方更深的黑暗——荒兽只是开始
- 伏笔植入：芽镇协议的不稳定性、远方黑暗的威胁、第二卷引子
- 出场人物：黎、五方使者、老药农
- 节奏控制：收束与展望，第一卷闭环，第二卷钩子

## 跨卷伏笔（本卷植入）
1. 被抹去的五方记忆——指向全书终极真相
2. 神秘的第六存在——全书终极反派/真相
3. 荒兽的来源——与洪荒之前的时代关联
4. 芽镇协议——后续卷中将被反复挑战
5. 黎体内的青木之种——完整形态尚未显现

## 节奏控制总览
- 第1-3章：慢热引入（日常→外部介入→多方交汇）
- 第4-6章：信息深化（能力探索→势力展开→五方齐聚）
- 第7-8章：转折与探索（主动选择→记忆深入→终极悬念）
- 第9-10章：高潮与收束（实战考验→格局确立→第二卷引子）
