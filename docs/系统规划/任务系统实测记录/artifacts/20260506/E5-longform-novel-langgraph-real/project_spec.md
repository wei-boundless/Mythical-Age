# 《雾港回声》长篇项目规格书

## 项目目标

启动长篇小说《雾港回声》正式立项，规划并执行总量 **1000000** 字的完整创作，分 **5卷** 逐卷推进，场景基底统一锚定于 **雾港**，确保全书结构完整、产物可追溯、验收有据。

## 固定设定锚点

- **标题**：雾港回声
- **主角**：林深
- **关键亲属**：林深的父亲
- **关键协作者**：老邱
- **关键线索物**：录音带
- **场景基底**：雾港
- **总规模**：1000000字
- **卷数**：5卷

- 章内允许弹性浮动（±2000字/章），但卷级字数必须闭合在200000字±5000字内。

## 5卷结构纲要

### 第一卷 · 雾起
林深因父亲失踪返回雾港。在父亲旧屋中发现一盒未标记的录音带。录音带内容指向七年前一桩被压下的港口事故。林深开始独自追查，在码头结识老邱。卷末：录音带中出现第二个人的声音，指向林深父亲并非单纯失踪。

### 第二卷 · 潮痕
林深与老邱建立协作关系。录音带逐段破译，牵扯出雾港三股势力：港口工会、旧船运家族、外来资本。父亲在录音中扮演的身份愈发模糊——是举报者还是参与者？卷末：第二盒录音带出现，由匿名者寄送。

### 第三卷 · 暗涌
双录音带交叉比对，父亲的双重身份浮现。林深与父亲关系进入核心拷问阶段。老邱的过往也被牵出——他与林深父亲曾是同事。卷末：关键证据指向雾港旧港区填海工程，但工程档案已被封存。

### 第四卷 · 雷暴
封存档案被强行打开。录音带最终段落解密，揭示事故真凶与掩盖链条。林深面临抉择：公开真相将摧毁雾港现存秩序，沉默则背叛父亲。老邱做出关键决断。卷末：真相公开，雾港进入震荡期。

### 第五卷 · 回声
震荡后的雾港重新寻找平衡。林深处理父亲遗留的全部未竟之事。录音带的最后一段空白被解读——那空白本身就是信息。林深离开或留下，雾港不再只是港口，而是他完成自我叙述的地方。

## 验收闸门

每卷完成后必须通过以下闸门方可进入下一卷：

1. **字数验收**：卷级正文字数在200000字±5000字范围内，逐章计数可追溯。
2. **线索闭合验收**：本卷关键线索物（录音带）的推进状态与 `_meta/clue_map.md` 记录一致，无断链、无矛盾。
3. **人物一致性验收**：主角林深、关键亲属（父亲）、关键协作者老邱的行为逻辑与 `_meta/character_bible.md` 一致，无OOC。
4. **场景一致性验收**：雾港地理、建筑、氛围描写与 `_meta/setting_bible.md` 无冲突。
5. **卷间接口验收**：卷末状态与下一卷卷首可无缝衔接，遗留线索已登记。

所有验收记录写入 `acceptance/` 目录对应文件，签字后方可推进。

## 禁止项

**禁止**在任一卷未实际完成时，伪造全本完成声明或以任何形式宣称"已完稿"。全书1000000字完成以5卷全部通过验收闸门为唯一判定标准，不接受预估、推断或口头声明替代正文产物。任何中间状态报告必须显式标注当前卷序、已完成字数和剩余字数，禁止模糊表述。
