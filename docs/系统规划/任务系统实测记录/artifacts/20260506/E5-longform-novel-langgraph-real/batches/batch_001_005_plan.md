## 批次目标

**批次一（第001章至第005章）** 的核心目标是：**世界初显·主角出场**。

具体而言，本批次需完成以下叙事任务：

1. **建立叙事基调**：以具体场景切入，让读者进入洪荒世界，感知其氛围与质感。避免宏大叙述，以可触摸的细节建立世界可信度。
2. **主角出场与初始状态锚定**：独立主角从微末中出场，展示其所在环境、身份、性格底色与潜在渴望。读者在5章结束时应对主角有清晰可感的印象。
3. **世界级事件的第一征兆**：一条来自远方的异常信号（东风异动/河流改道/星象偏移等）作为五荒深层矛盾的第一个征兆进入叙事，成为推动主角启程的外部动力。
4. **五荒元素的初步落地**：河伯背景种子（第3章渡口场景）与勾芒背景种子（第4章青木之域边缘）在本批次内自然融入叙事，不堆砌。
5. **主角身份锚定**：主角在第5章获得或确认自己的"名字"（在洪荒语境中名字具有力量），完成"我是谁"的第一层确认。
6. **伏笔体系启动**：投放伏笔#1（异常信号来源，跨卷）、伏笔#2（主角未被察觉的特质，卷内回收）、伏笔#3（渡口祭辞隐秘信息，跨卷）、伏笔#4（主角名字中的渊源，跨卷）。

#### 冲突推进
- 本章冲突为**首次明确对抗**：前4章积累的张力（异常信号→选择启程→接触更广阔世界→感知域枢力量）在此转化为具体的、可命名的冲突。
- 冲突规模应控制在"小型"——不是五荒势力之间的对抗，而是主角个人层面的考验。
- 主角在冲突中展现的初始反应模式将成为后续成长的参照基线。
- 冲突的解决方式暗示主角的核心特质（伏笔#2在此获得第一次隐性推进，但不回收）。

#### 关键意象
- **名字**：洪荒语境中名字具有力量——名字不是标签，而是存在的确认。主角名字的获得/确认应具有仪式感。
- **第一个对抗**：冲突中的具体意象（武器/自然力/对手的特征）应具有辨识度，可在后续章节中被回溯。
- **回望**：章节结尾主角回望来路的空间意象——已走过的路、已渡过的河、已穿过的林——作为"成长"的空间隐喻。
- **前方**：主角面向的方向——暗示第二卷的方向，但不具体揭示。

## 风险与注意事项

1. **开篇节奏**：第1章需避免"慢热"——日常场景应有足够的细节质感支撑，但不能拖沓。异常信号应在章节前半段引入。
2. **主角独立性**：主角的决策和行动必须始终源于自身动机，不能出现"被域枢召唤"或"命运指引"式的被动启程。
3. **名字的仪式感**：第5章名字的获得/确认是主角弧线的第一个关键节点，需有足够的情感重量，但不能过度煽情。
4. **冲突规模控制**：第5章的第一个明确冲突必须控制在"小型"——个人层面的对抗，不能过早引入五荒势力之间的冲突。
5. **伏笔隐蔽性**：四条伏笔的投放必须融入场景和人物行为，不能以"叙述者告知"的方式暴露。伏笔#2尤其需要轻描淡写。
6. **五荒平衡**：本批次仅落地河伯与勾芒两个背景种子，其余三个留待批次二。不要在批次一中试图塞入所有五荒元素。
