# 玩法与剧情设计文档

## 核心玩法
- **俯视角实时动作**：玩家使用键盘（WASD 或方向键）控制角色在房间内移动，鼠标点击或键盘按键攻击。
- **房间探索**：每层由若干矩形房间组成，门相连，房间随机生成敌人、宝箱、祭坛等。进入新房间时门关闭，击败所有敌人后门打开并可能掉落升级选项。
- **敌人AI**：敌人有巡逻、追击、攻击三种状态。近战敌人冲向玩家，远程敌人保持距离射击。精英敌人拥有特殊技能（如冲锋、召唤小怪）。
- **升级系统**：击败敌人获得经验值。升级时弹出三选一选项，可从“提升攻击力”“提升生命值”“学习新技能（如冲刺、范围攻击、吸血）”中选择。效果即时应用。
- **Boss战**：每层末尾房间为Boss房，Boss拥有独特攻击模式（弹幕、冲锋、召唤、AOE等），击败后掉落钥匙，可进入下一层。
- **生命与死亡**：玩家有生命条，受到伤害扣减，生命为0游戏结束，显示统计信息。通关最终Boss后显示胜利画面。
- **HUD**：左上显示生命条、经验值、等级；右上显示当前层数；底部技能栏。

## 敌人类型
1. **骷髅兵**（近战）：基础近战，移动慢，攻击低。
2. **骷髅弓箭手**（远程）：与玩家保持距离，发射直线箭矢。
3. **暗影刺客**（精英）：高移速，周期性隐身突袭。
4. **Boss - 亡灵骑士**（第1层）：近战为主，附带冲锋技能。
5. **Boss - 大法师**（第2层）：远程弹幕，召唤小怪。
6. **Boss - 石像鬼**（第3层）：空中单位，向下俯冲攻击。
7. **Boss - 火焰领主**（第4层）：范围火焰攻击，留下燃烧地面。
8. **Boss - 黑暗君主**（第5层）：混合攻击，二阶段变身。

## 升级选项
- **力量祝福**：攻击力 +5
- **生命之泉**：最大生命值 +20
- **疾步**：移动速度 +10%
- **嗜血**：击杀敌人回复少量生命
- **闪电链**：攻击有概率连锁到附近敌人
- **护盾**：每层获得一个临时护盾

## Boss机制
每个Boss拥有多个阶段，生命低于50%进入狂暴状态，攻击频率提高。特殊攻击有地面提示，玩家需走位躲避。击败Boss回复部分生命，并获得升级选项。

## 剧情背景
古代亡灵塔封印着黑暗君主，每百年需要用勇者灵魂加固封印。主角是一名无名的冒险者，为了拯救被诅咒的村庄，毅然踏入五层亡灵塔，击败层层守护者，最终挑战黑暗君主。塔内遍布残破的盔甲、古老的壁画和飘荡的幽灵，通过环境叙事传递历史。

## 五层地下塔探索目标
- **第一层·幽暗墓穴**：学习基础操作，击败亡灵骑士。
- **第二层·魔法实验室**：面对大法师的弹幕，考验走位。
- **第三层·飞石大厅**：应对石像鬼的俯冲攻击。
- **第四层·岩浆核心**：在火焰领主留下的燃烧区域中战斗。
- **第五层·黑暗王座**：最终Boss战，黑暗君主。
