# 项目简报：2D 俯视角 Roguelike 地下塔

## 项目目标
创建一个可运行、可测试、可迭代的浏览器端 2D 俯视角 Roguelike 游戏垂直切片。玩家将探索五层程序化生成的地下塔，与敌人战斗，获取升级，击败楼层 Boss，最终通关或死亡。

## 范围
- **核心玩法**：实时俯视角移动、近战/远程攻击、敌人 AI、房间生成、物品/升级拾取、HUD 显示生命/经验/等级。
- **内容**：5 层地下塔，每层包含若干房间，敌人类型至少 3 种（近战、远程、精英），每层末尾有 Boss。
- **升级系统**：击杀敌人获得经验，升级时可选择属性提升或新技能。
- **胜负条件**：生命值降为 0 游戏结束，击败最终 Boss 通关。
- **叙事元素**：简短背景故事，通过环境叙事和击杀 Boss 后的文本推进。

## 约束
- 完全在浏览器中运行，无需后端服务器。
- 使用纯 HTML/CSS/JavaScript，不依赖大型框架，便于迭代和测试。
- 资源全部由代码生成或本地图像文件提供，无需联网。
- 游戏文件应便于在本地文件系统中打开，或通过简单 HTTP 服务器运行。

## 目标平台
- 现代浏览器（Chrome、Firefox、Edge 等），支持 Canvas 2D 渲染。
- 桌面分辨率为主，基础响应式设计。

## 技术栈
- **HTML5 Canvas**：渲染游戏画面。
- **JavaScript (ES6+)**：游戏逻辑、输入处理、程序化生成。
- **CSS**：HUD 和菜单样式。
- **本地图像文件**：角色和背景等位图资产（PNG）。
