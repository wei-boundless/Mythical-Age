## Canonical Memory
2025年AI治理报告《回归现实主义》的核心结论是：全球AI治理正在经历'去理想化'进程，治理重心从防范假设性末日风险转向释放产业潜能。报告第二部分主张模型治理走'小切口、轻义务'路线，以加州SB53为典型样本：监管对象收窄至前沿开发者（训练算力>10²⁶ FLOPS且年收入>5亿美元），义务轻量（透明度报告仅需基础信息），与欧盟全面细颗粒监管路径形成明确对比。第二部分核心约束可归纳为两个维度：路径约束（治理对象精准，只抓最顶层的几款模型）和方法约束（治理工具可操作，透明度、可解释性、对齐是三大落地抓手）。欧盟AI法案2025年进入落地实施阶段：禁止性AI首批义务自2025年2月2日起生效，通用人工智能模型（GPAI）相关义务自2025年8月起生效，欧盟委员会同步发布GPAI指南、行为准则等合规工具，并搭建AI Office组织机制配合成员国监管体系推进。
